# Ajrasakha Frontend

A React-based web application for monitoring crop health using satellite data and AI-generated insights.

## Features

* Dashboard for crop health overview
* Interactive map for region monitoring
* AI-generated crop health insights
* Alert management system
* Farmer and region management
* Responsive design for desktop and mobile

## Tech Stack

* React
* TypeScript
* Vite
* React Router
* Axios
* Leaflet

## Installation

```bash
npm install
```

## Run the Application

```bash
npm run dev
```

The application will start at:

```bash
http://localhost:5173
``

## Main Pages

* Dashboard
* Map View
* Heatmap
* Insights
* Alerts
* Farmers
* Admin Dashboard
* Operations

## API Configuration

Update the backend API URL inside:

```text
src/services/api.ts
```

## Build for Production

```bash
npm run build
```
